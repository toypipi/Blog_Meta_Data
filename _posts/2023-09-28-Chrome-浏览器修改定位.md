# 插件

Chrome 应用商店安装 [Location Guard](https://chrome.google.com/webstore/detail/location-guard/cfohepagpmnodfdmjliccbbigdkfcgia?hl=zh-CN)

# Chrome 开发者工具

F12 打开开发者工具 -> 点击右上角竖排 三个点 -> More tools ->Sensors -> Location -> 选择地址或输入定位
