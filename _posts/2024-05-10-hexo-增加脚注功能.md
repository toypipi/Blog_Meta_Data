1.打开 hexo 官方插件页面 [Plugins](https://hexo.io/plugins/)  
2.搜索 `footnotes` 并安装 [hexo-reference](https://github.com/kchen0x/hexo-reference)

```
npm install hexo-reference --save
```

配置根目录 `_config.yml` 文件，新增一下内容：

```
plugins:
  - hexo-reference
```

<!-- more -->

3.效果

鼠标移动到数字[^1]上，即可显示脚注内容。

# 注意事项

1. 脚注内容不能包含 `""` 符号，否则会报错。
