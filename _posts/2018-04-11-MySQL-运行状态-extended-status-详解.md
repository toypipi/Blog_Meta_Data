使用以下命令查看 MySQL 详细运行状态：
```
mysqladmin -u root -p extended-status
```

不同版本的 MySQL 输出的参数个数不同。以下为从 MySQL 5.7 版本官方文档截取的输出，各个参数的详细解释见下，中文含义后续用到哪个再去翻译吧。

{% endnote %}
