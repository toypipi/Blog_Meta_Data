1. chrome 安装插件 JetBrains IDE Support。
2. WebStorm 打开 Settings >> Buid,Execution,Deployment >> Debugger >> live Edit ,勾选 Use JetBrains IDE Support extension for debugging and live Edit 选项。
3. done!
